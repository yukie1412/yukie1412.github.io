Variables

What is a Variable

In simple terms, a variable is anything that stores value.

Types of Variables

There are 3 basic types of variables: string, boolean, number.

Defining a Variable in JavaScript

All variables in JavaScript are defined by the keyword var, follow by the variable name (and assignment), and end with a semi-colon ;

Remember:

  • variable names should be camel case: var firstName = "Jennifer";
  • variable names should be descriptive: var sayHello = "Hello";
  • variable names must start with a letter.
