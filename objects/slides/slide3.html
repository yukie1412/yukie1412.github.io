What is an Object

Definition

A set of key-value pairs, enclosed in { and }:

var cat = {
	petName: "name",
	isFemale: true,
	happiness: 30
};

cat is an object with keys petName, isFemale, and happiness. Its values are name, true, and 30 respectively.

Key

Any string. Be descriptive, avoid spaces. Keys must be unique within the object.

Value

Any supported JavaScript variable types.
