null

Mozilla defines null as "Intentional absence of any object value."

null is a special type of object that must be explicitly defined:

var nullObj = null;

undefined

Mozilla defines undefined as "a property of the global object."

undefined is the default value for any variable without a value:

  • Consider var num = 5;, num has the value of 5
  • If we rewrite it as var num;, then num has the value of undefined
