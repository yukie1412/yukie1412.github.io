Access The Value

Consider the code below:

var obj = {
	"first property": "abc",
	secondProperty: 123
};
var key = "secondProperty";

Dot Notation

Recommended syntax when the key is a valid JavaScript identifier (how you would define a variable).
Basically a dot (.) is sandwiched between the object and the key.

  • obj.secondProperty gives you 123

Bracket Notation

Use this syntax when the key is a variable, or an invalid JavaScript identifier.
The key is enclosed in the square brackets ([ & ]), immediately after the object.

  • obj["first property"] gives you "abc"
  • obj[key] gives you 123
