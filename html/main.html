Basic HTML Structure

<!DOCTYPE HTML>

<html>
    <head>...</head>

    <body>...</body>
</html>
<!DOCTYPE HTML> is a must-have first-line.

<html> is the beginning of an HTML document.
<head> is the container for all information about the page.

<body> is the container for all content about the page.

head document information

<title>page title</title>
<meta charset="UTF-8">
<meta name="description" content="brief site description">
<meta name="keywords" content="comma-separated,keywords">
<meta name="author" content="your name">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<base href="base location of links on the page"></base>
<style>CSS Styles</style>
<link href="link to CSS file" ref="stylesheet">
<script src="link to JavaScript file" text="text/javascript"></script>

body document content

<header>...</header>
<nav>...</nav>
<section>...</section>
<article>...</article>
<aside>...</aside>
<footer>...</footer>

<button>...</button>
<a href="link to another page">...</a>
<p>...</p>

<ul>
    <li>...</li>
</ul>

<ol>
    <li>...</li>
</ol>
<header> contains the header content.
<nav> contains the navigation menu.
<section> represents a generic section of a document.
<article> is a self-contained composition in a document.
<aside> contains miscellaneous information.
<footer> contains the footer content.

Sample Structure

<!DOCTYPE HTML>

<html>
    <head>
        <title>Girls Who Code</title>
        <meta charset="UTF-8">
        <meta name="description" content="Girls Who Code Saturday Club at Millbrae Library">
        <meta name="keywords" content="GWC, Millbrae Library">
        <meta name="author" content="Jennifer,Huimin,Donna,Kaitlyn,Samantha">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>

    <body>
        <header>
            <nav>
                <ul>
                    <li>Link 1</li>
                    <li>Link 2</li>
                </ul>
            </nav>
        </header>

        <section>
            <article>
                <h2>Article 1</h2>
                <p>
                    Paragraph for article 1
                    <a href="url">Link</a>
                </p>
            </article>

            <article>
                <h2>Article 2</h2>
                <p>
                    Paragraph for article 2
                    <button>button</button>
                </p>
            </article>
        </section>

        <aside>
            <h2>Aside</h2>
            <p>Paragraph for article 1</p>
        </aside>

        <footer>
            <p>Copyright</p>
        </footer>
    </body>
</html>
