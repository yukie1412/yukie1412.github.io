Basic Operators

There are 5 basic operators, very similar to math.

Increments & Decrements

A lazy alternative to add 1 or subtract 1.

Assignment Operators

A lazy alternative to update a number.

Short-cut Explained

x++; is the same as x += 1; is the same as x = x + 1;

x--; is the same as x -= 1; is the same as x = x - 1;

x += y; is the same as x = x + y;

x -= y; is the same as x = x - y;

x *= y; is the same as x = x * y;

x /= y; is the same as x = x / y;

Please note that the equal sign = means assignment, not equals to.

Imagine a variable z, then x += y is the same as z = x + y; followed by x = z;.

Imagine Packing a Suitcase

The suitcase is x, and clothes are y, and the process of adding clothes into the suitcase is x += y;.

Everytime you add (or remove) a cloth from the suitcase, the weight of the suitcase changes.

Similarly, when you update x with the short-cut operations, value of x changes.
